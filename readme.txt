Программа предназначена для снятия точных образов с флешек
и дискет, а также для последующей записи образов обратно.

Так же, возможно снятие образов с жестких дисков, и CD\DVD
устрйств.

!!!Примечание: Образы снятые с CD\DVD можно использовать как
виртуальный диск, например в Alcahol'е, или записывать их
используя NERO. Сама программа FleshImager не умеет производить
записи на CD\DVD.

!!!Примечание: Снятие образа с CD\DVD происходит по последней
дорожке записи.
